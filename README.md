## 🎭 Что это вообще такое?

**Oskolki** — это мобильное приложение, которое превращает скучную реальность в AR-приключение! 🎮

Представь: ты идёшь по улице, открываешь камеру — и БАМ! 💥 Повсюду фрагменты (осколки) с текстами, фото и аудио от других пользователей. Как Pokemon GO, только с памятью и без покемонов (ну почти).

### 🌟 Особенности (которые мы гордо называем "фичами"):

- 🗺️ **Яндекс.Карты** — потому что Google Maps уже всех достали
- 📸 **AR-метки** — фрагменты прямо перед носом через камеру
- 🎯 **Умный размер** — метки растут когда близко (как твоя тревога перед дедлайном)
- 📍 **Геолокация** — находим осколки в радиусе 50 метров
- 🔐 **Авторизация** — входи и регистрируйся, как настоящий хакер
- 🎵 **Аудио-сообщения** — слушай, что напели другие (надеемся, они в тональности)
- 📷 **Фотографии** — смотри фотки мест, где люди оставили след (или просто перекусили)

## 🛠️ Технологический стек (или "на чём это работает")

```
🎨 Frontend:    Kotlin + CameraX + AR Manager
🌍 Maps:        Yandex MapKit (русский стайл!)
🌐 Backend:     Retrofit + OkHttp
📦 Build:       Gradle (который вечно что-то качает)
🎭 Sensors:     TYPE_ROTATION_VECTOR (магия с гироскопом)
```

## 🚀 Как запустить (спойлер: непросто)

1. **Клонируй репозиторий** (ты же это умеешь, правда?)
   ```bash
   git clone https://github.com/твой-юзернейм/Oskolki.git
   ```

2. **Открой в Android Studio** (пока он не завис на 99% индексации)

3. **Добавь свои API ключи** в `secrets.properties` (не коммить его, будь умным!):
   ```properties
   YANDEX_MAPS_API_KEY=твой_супер_секретный_ключ
   BASE_URL=https://твой-сервер.рф
   ```

4. **Жми Run** и молись, чтобы Gradle не решил обновиться

## 🎪 Архитектура (или "как мы запутались")
